## 📌 About
Этот репозиторий содержит решения домашних заданий и лабораторных работ по курсу Python в Университете Иннополис. Задачи охватывают широкий спектр тем: от базового синтаксиса до алгоритмизации и решения прикладных задач.

## ⚙️ Состав задач
- **Основы Python:** Работа со списками, словарями, циклами и функциями (недели 2-4).
- **Алгоритмические упражнения:** Задачи на сортировку, поиск и динамическое программирование (недели 5-9).
- **Продвинутые темы:** Работа с файлами, исключениями и модулями (недели 10-12).

## 🛠 Стек технологий
- **Язык:** Python 3
- **Парадигма:** Процедурное программирование, функциональный стиль.
